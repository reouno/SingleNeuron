# SingleNeuron

Installation
============

Just input following commands in terminal

$ git clone https://github.com/reouno/SingleNeuron.git

$ cd SingleNeuron

$ stack build

Execution
=========

$ stack runghc app/Main.hs

or

$ stack exec SingleNeuron-exe
